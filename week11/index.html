What is true and false?

"Falsy" Values

  • Boolean false: false
  • The number zero: 0
  • An empty string: ''
  • A variable with no value

"Truthy Values"

  • Boolean true: false
  • The number zero: 42, -42, etc.
  • Strings: 'Hello world'
  • Anything that's not "falsy"

Loops

  • A loop checks a condition and if it is true, repeats until the condition is false.
  • Helpful for iterating over arrays and node lists.
  • Three types
    • for
    • while
    • do while

Document Object Model (DOM)

  • The DOM is how the browser understands your HTML code in outline form.
  • See the lecture slides for more information and examples.
