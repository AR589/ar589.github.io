Designing with Relationships

When designing think about the relationship of elements in your design. When possible try to use flexible units to make relationships responsive.

  • Science on Screen
  • Grove

Units of Measurement

Fixed Units

  • px
    a unit calculated by the browser
  • em
    a unit relative to the font size of the element
  • rem
    a unit relative to the font size of the html element

Flexible Units

  • %
    a percentage of the parent element
  • vw
    a percentage of the viewport width
  • vh
    a percentage of the viewport height
